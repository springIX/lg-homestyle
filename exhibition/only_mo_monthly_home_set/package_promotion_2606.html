    50만원 이상 결제 시 할인 제공

    (매월 1인당 누적 100만원, 횟수 제한 없음)

  • 10% 결제 혜택

    LGE.COM 신한카드로
    50만원 이상 결제 시 할인 제공

    (플러스 서비스 이용 시 10% 캐시백 적용)

  • 무이자 혜택

    신한/현대/KB국민/롯데/하나/우리카드로
    100만원 이상 결제 시 적용

    (플러스 서비스 이용 시 10% 캐시백 적용)

    네이버페이 15만원 이상 결제 시,
    상세 내용 하단 유의사항 참고

  • 정확한 혜택은 상품 상세정보및 최하단 프로모션 유의 사항을 확인해주세요.
