네 단계로 끝나는 우리 집 자랑

  1. 커뮤니티 > “내 공간 자랑”에 글쓰기

    커뮤니티 게시판에 새 글 작성 시작

  2. 미션 태그 선택하기

    5개 미션 태그 중 내 공간에 맞는 태그 선택
    (복수 가능)

  3. 사진, 글, 태그 업로드 하기

    사진 3장 이상(공간컷 1장 필수) + 글 최소 100자
    이상 업로드 + 미션 태그 1개 이상 입력

  4. ‘이벤트 응모하기’버튼 클릭

    작성 완료 후 응모 버튼을 눌러야 응모 완료!
